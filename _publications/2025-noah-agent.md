Status: Preprint, version 0.1, in preparation.

Noah-Agent is a multi-expert AI agent framework for automated parameterization and validation of large-scale Fortran climate models.

## Approach
- Multiple specialized agents coordinate to parameterize and validate large Fortran model codebases.
- The framework automates steps that are otherwise done by hand, such as parameter selection and validation against reference behavior.

## Relation to Other Work
Noah-Agent connects to the broader Noah-MP land surface modeling effort and to [ESM-bench](/publication/2026-esm-bench), which evaluates whether AI agents understand Earth System Model physics and code.
