我是怎么用 Claude 工作的

五个真实使用场景，每页一个 demo
Koutian Wu2026-05-09
场景 1 / 5

用 Claude Code 写代码、提 PR

  • 在终端里直接跑 claude，让它读仓库、改文件、跑测试
  • 改完自动 git commit，再用 gh pr create 开 PR
  • 从「输入需求」到「PR 链接」，不用离开命令行
Demo: 现场让它修一个 bug 并提 PR
场景 2 / 5

用 Claude Code Skills 封装常用流程

  • 把重复的判断流程写成 skill，按名字一键调用
  • 例如：buffett-checkhedge-reviewredteam
  • 团队共享同一套 skill，结果可复现
Demo: 触发一个 skill，看它如何按步骤执行
场景 3 / 5

给 Tacit 搭一个内部知识网站

  • 把公司内部文档、流程、决策记录作为上下文导入 Claude
  • 员工用自然语言提问，直接拿到带引用的答案
  • 新人 onboarding、跨团队信息同步、合规问答都能用
Demo: 提一个内部问题，看检索 + 回答效果
场景 4 / 5

用 Claude 浏览器扩展做 Computer Use

  • 让 Claude 直接在浏览器里点按钮、填表单
  • 典型用例：在 LinkedIn 上批量发连接请求 + 个性化备注
  • 把「人工 + 重复点击」的活儿交给 agent
Demo: 现场打开 LinkedIn，让它发一条连接请求
场景 5 / 5

Skills 实例：投资组合分析

  • 本仓库 KW-portfolio：YAML 持仓 + skill 自动出报告
  • 每个快照生成 HTML dashboard，含对冲、red team、Buffett 检查
  • 同一套 skill 也能用于客户资产、内部财务复盘
Demo: 打开 reports/20260508.html
