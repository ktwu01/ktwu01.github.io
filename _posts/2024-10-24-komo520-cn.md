这是一个手工打造的网站小宇宙：每一个网页、气球、点击和每一行代码都在说“我爱你”——只为 Momo 而设计。[访问在线站点](https://ktwu01.github.io/komo520/)

> 作者：[Koutian Wu](https://www.linkedin.com/in/ktwu01/)；[GitHub: ktwu01](https://github.com/ktwu01/)

## 🌸 项目概览

这个 GitHub Pages 合集包含了几十个互动式小网页，围绕特殊纪念日、心情、周年、情书和小游戏展开。每一个页面，都是藏在我们宇宙里的一个**数字糖果罐**。

**功能亮点**：
- 👆 互动元素（比如挠猫猫）
- 💬 模拟对话小剧场（Koko 作为 NPC）
- 🎈 爱心气球、节日特效、倒计时计时器
- 🧠 小测验和模拟器（Koko 的 JS 脑筋急转弯）
- 🖼️ 充满回忆和彩蛋的插画

## 💻 技术栈

- `HTML` + `CSS` + `JavaScript`
- 框架：`Vue.js`, `jQuery`, `Zepto.js`
- 样式：手写 CSS / SASS / Tailwind utilities
- 构建工具：`Gulp`（用于 live chat）
- 托管：GitHub Pages

## 🗂️ 项目内容

| 名称 | 说明 |
|------|------|
| `Tickle-Momo-cat` | 轻轻戳一戳小猫，她会说甜甜的话 |
| `Komo-LoveBalloon` | 为特殊时刻准备的互动气球小游戏 |
| `HongMomoSimulator` | 模拟 Momo 日常心情变化 |
| `Komo-live-chat` | Momo 可以和 AI Koko 聊天的聊天室 |
| `SmallPlay` | 一组小测试与惊喜合集 |
| `Anniversary Pages` | 为周年纪念专门制作的页面 🗓️ |

## 🛠️ 如何运行

这是一个静态网站，不需要额外配置：

```bash
git clone https://github.com/ktwu01/komo520.git
cd komo520
# 打开 index.html 或任意子目录中的 HTML 文件
```

也可以直接访问： [ktwu01.github.io/komo520/](https://ktwu01.github.io/komo520/)
