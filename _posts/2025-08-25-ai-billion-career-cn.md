这是一个基于 React、Vite 和 Supabase 构建的综合工具，帮助个人规划并导航自己在 AI 行业中的职业发展路径。

> 作者：[Koutian Wu](https://www.linkedin.com/in/ktwu01/)；[GitHub: ktwu01](https://github.com/ktwu01/)

## 项目结构

项目主要由以下目录组成：

- `ai-career-dashboard/`：主前端应用，使用 React、Vite 和 Supabase 构建。更详细信息请查看 [dashboard README](./ai-career-dashboard/README.md)
- `code/`：用于 AI 职业数据分析与处理的 Python 脚本
- `docs/`：支撑该项目的一组研究文档、分析报告与规划材料
- `extract/`：JSON 格式的原始数据文件，供 dashboard 和分析脚本使用
- `supabase/`：Supabase 后端的配置与迁移脚本

## 快速开始

若要启动主应用，请先进入 `ai-career-dashboard` 目录，并按照其中 `README.md` 的说明操作：

```bash
cd ai-career-dashboard
npm install
npm run dev
```
