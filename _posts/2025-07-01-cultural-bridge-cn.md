一个基于 Web 的平台，用于促进跨文化交流与理解。

## 本地部署

### 后端配置

```bash
cd cultural-bridge
source venv/bin/activate
pip3 install -r requirements.txt
python3 main.py
```

后端会启动在 `http://localhost:8000`

### 前端

直接在浏览器中右键预览对应的 HTML 文件即可。

![Preview](image.png)

## 故障排查

如果遇到报错，可以按浏览器 `F12` 查看错误信息，再结合 GPT/Cursor 一起排查。

## 技术栈

- **Backend**: Python, FastAPI
- **Frontend**: HTML/CSS/JavaScript
- **Deployment**: Local server
