团队里事情太多、问题太多、半截决定散在 Slack 串和会议纪要里飘来飘去，我自己反复试下来，唯一能阻止这种混乱慢慢腐烂的办法，就是把所有真正值钱的知识强行收进同一个地方，那就是 GitHub Repo，其它任何地方都只能当临时草稿纸。

> 作者：[Koutian Wu](https://www.linkedin.com/in/ktwu01/)；[GitHub: ktwu01](https://github.com/ktwu01/)

一样的失败模式，我在课题组、在创业团队、在志愿者组织里反复见过。有人在会上问了一个挺尖锐的问题，资深成员当场答得很漂亮，三个人点头认同，结果一周之后新来的成员进来，问的是同一个问题。那个答案在房间里活过一次就死了，因为没人把它写到「下一个人会自然去查」的地方。把这件事乘以一整年，你就得到一个把「在动」当成「在进步」的团队。

解决方案不是开更多会。解决方案是只有一个真相之地。

对一个工程或者研究团队来说，这个唯一的真相之地就是 GitHub Repo。不是 Notion，不是飞书文档，不是某个共享的 Google Drive，也不是某人侧边栏置顶的 Slack 频道。是这个 Repo，它的 README，它的 docs 目录，它的 issues，它的 pull requests，它的 commit history。其它一切要么是喂数据进 Repo 的临时管道，要么就根本不算数。

这个地方就是最好的，单一的最好的地方。

我知道一个很自然的反驳。有人会说 Repo 是放代码的，文档和讨论应该去更花哨的工具里，那种有评论、有 emoji、有嵌套页面的地方。我懂这个直觉，Notion 确实好看，Slack 确实快。但这些工具优化的那个东西，写起来轻松，恰恰是它们不适合当真相之地的原因。写起来轻松意味着同一件事容易在两个地方各写一遍，然后你就有两个版本的答案，谁也分不清哪个是当下有效的。

Repo 的胜出在于它对碎片化是有敌意的。有一条 main 分支。有一张 commit 图记录了谁在什么时候改了什么。有 PR 强迫你在变更进入正史之前先经过一轮审视。这种摩擦本身就是功能。当一件事必须穿过这层摩擦才能进 Repo，团队会隐隐学到，这里才是「主张变成事实」的地方。

放进 Repo 的也不只是代码。README 是前门，它告诉新成员这个项目是什么，怎么在五分钟内跑起来。docs 目录放设计决策，放「为什么这样做」的笔记，放那些每个人都会重新踩一遍的坑。issues 跟踪那些悬而未决的问题，带着标签、负责人、清晰的状态。PR 是大家用书面方式讨论提案的地方，配着一份你真的能看懂的 diff。甚至 commit message，如果你认真对待它，它就是这个团队思考方式的连续叙事。

我跟合作者讲，决定如果没进 Repo，就等于没发生过。这话有点狠，本来就是要狠。因为另一种文化是，房间里嗓门最大的人拥有真相，这种文化会惩罚新成员，惩罚不爱发言的贡献者，惩罚在另一个时区、根本没参加那次会议的人。Repo 是这件事里最大的平权工具，谁都能读，谁都能开 issue，谁都能提 PR，所有工作都以一种人人可审视的形式说话。

我一直觉得，背后还有更深的一层原因。团队不是一个单独的脑子，而是一张相互重叠得不太严丝合缝的脑子的网。每个人脑子里的项目版本都略有不同。Repo 是唯一一个能让这些版本被强行对齐的地方，因为它是唯一一个把「对齐」逼成单一线性历史的地方。Slack 让你可以无限期地活在平行宇宙里。Repo 逼你 merge。

落到日常上具体长啥样呢？有人提一个新方向，要去开一个 issue 把提案写清楚，而不是丢一条 Slack。会议产生的决定，必须在散会前由某人写进 Repo 里对应的文档，不然这个决定就只是暂时的。新成员加入，被指着 README 让他读到自己有问题为止，他冒出来的问题再变成 issue，反过来让 README 变好。线上出了状况，事故复盘进 Repo，而不是进某人 Drive 里那个六个月后就再也找不到的 Google 文档。

这事不是没成本的。写下来比说出来慢。一开始一定会有人抗拒，尤其是那种「权威感」建立在「我知道一些别人不知道的事」上的人。这种抗拒本身就是信号，它告诉你团队里谁从碎片化里得到了好处。从口头文化转向「Repo 即真相」的文化，悄悄地是一次权力的迁移，从非正式的、靠脑容量记着的影响力，转向书面的、可审阅的、可被反驳的主张。有些团队迈不过这道坎，那它们要么一直小，要么一直乱。

迈过去的那些团队会拿到一份不错的回报。新人一天而不是一个月就能上手项目。即使有人离开，记忆也还在。可以诚实地回头看上个季度的决定，问一句「我们是不是说到做到了」，因为白纸黑字就在那儿。最后这一点是大多数团队最不敢做的事，因为诚实的回顾需要诚实的记录，而 Repo 是为数不多没办法被悄悄改成「过去比当时显得更体面」的地方之一。

选 Repo。让它成为唯一。其它地方都是草稿纸。
