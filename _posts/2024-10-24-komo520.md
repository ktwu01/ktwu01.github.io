## 🌸 Project Overview

This GitHub Pages collection features dozens of interactive mini-webpages dedicated to special occasions, moods, anniversaries, love letters, and games. Each page is a **digital candy jar** hidden in our universe.

✨ **Features**:
- 👆 Interactive elements (like tickling the cat)
- 💬 Simulated dialogue mini-theater (Koko as NPC)
- 🎈 Love balloons, holiday effects, countdown timers
- 🧠 Mini quizzes and simulators (Koko's JS brain teasers)
- 🖼️ Illustrations full of our memories and easter eggs

## 💻 Tech Stack

- `HTML` + `CSS` + `JavaScript`
- Frameworks: `Vue.js`, `jQuery`, `Zepto.js`
- Styling: Hand-written CSS / SASS / Tailwind utilities
- Build Tools: `Gulp` (for live chat)
- Hosting: GitHub Pages

## 🛠️ How to Run

This is a static website, no additional configuration needed:

```bash
git clone https://github.com/ktwu01/komo520.git
cd komo520
# Open index.html or any HTML file in subdirectories
```

Or visit directly: [ktwu01.github.io/komo520/](https://ktwu01.github.io/komo520/)
